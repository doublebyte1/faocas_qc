Detailed Description

Custom Input Control class.

This control standardizes the input of catch in a UI composed of column tabs; Each tab corresponds to a different method of inputing catch, and totals are updated in the case of catch composed by a binomio (for instance unit and weight); In this way, we hope to achieve a user friendly UI that allow us to collect as much information as possible and at the same time provide some kind of validation for the user; n.b.: at this moment, unit conversions are not supported and therefore the units of the total and of the other pages have to match, in order to benefit from the calculations!
